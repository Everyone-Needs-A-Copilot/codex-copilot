# Code of Conduct

## Our Standard

This project aims to be useful, direct, and technically serious.

Participants are expected to:

- act respectfully
- critique ideas and implementations without attacking people
- keep discussion grounded in evidence and reproducible behavior
- avoid harassment, discrimination, and personal attacks

## Scope

This applies within project spaces, including issues, pull requests, code review, and discussions tied to this repository.

## Enforcement

Project maintainers may remove content, close discussions, or restrict participation for conduct that undermines a productive and respectful working environment.
